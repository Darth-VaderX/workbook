# Оператор in
____
Оператор __in__ проверят есть ли свойство у объекта или массива и возвращает true либо false.
```
let obj = {
	name: 'Nick',
	age:  23,
};

console.log('name' in obj); // true
console.log('age'  in obj); // true
console.log('test' in obj); // false
```
